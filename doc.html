ZenSL Optimizer

Optimize Your Computer For Gaming

What is ZenSL?

ZenSL is a Windows optimizer that modifies Windows settings, game settings, and creates a smooth OS experience!

Windows Settings

Windows Settings App

  • Core Isolation (Microsoft Windows Defender): Set OFF
  • Gaming Mode: Set ON
  • Background Apps (Privacy Settings): Set OFF

Windows PowerPlan (Control Panel Settings => Power Management)

  • Set GUID Scheme 'Ultimate Performance' Active
  • Optimize Settings for Performance

Windows Task Manager Autostart-Apps (Microsoft Supplied Apps Only)

  • Microsoft Teams
  • Security Health System Tray
  • DevHome
  • Smartphone Link
  • Mobile Devices
  • Microsoft Edge
  • XBOX App Services

Windows Advanced Settings

  • Manage VRAM: Set to best amount (calculated by Python script)
  • Customize UI Features: Leave unnecessary things OFF

Optimize Network

  • Set settings for the used Wi-Fi adapter for optimal gaming network speeds

Hardware Optimizations

GPU

NVIDIA

  • General Optimizations: Includes Nvidia Control Panel settings and enabling discrete graphics mode.
  • RTX Series Optimizations: Focuses on overclocking with MSI Afterburner and setting up Nvidia GeForce Experience for performance.
  • GTX Series Optimizations: Similar to RTX but with adjustments specific to GTX series in Nvidia GeForce Experience.

AMD

  • General Optimizations: Involves AMD Adrenaline settings like performance mode and graphics adjustments.
  • Specific Series Optimizations: Tailored optimizations for RX 5000, RX Vega, RX 400/500, and RX 300 series.

INTEL

  • Integrated Graphics Processing Units (IGPUs): Includes settings for Iris and Arc graphics with a focus on performance and custom profiles.
  • Intel ARC Series Optimizations: Details on performance settings, custom profiles, and ray tracing features for Intel Arc GPUs.
